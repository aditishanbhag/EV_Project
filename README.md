# ⚡ EV ChargePro

A web-based Electric Vehicle Charging Station Management System built with Flask and MySQL. Users can find nearby charging stations, book slots, manage their EVs, and pay for sessions — all from a sleek dark-themed dashboard.

---

## Features

- User registration and login
- Dashboard with stats on sessions, energy used, and total cost
- Add and manage multiple EVs
- Browse and book available charging slots
- Interactive GPS map (Leaflet.js) to find the nearest station
- Charging session history
- Razorpay payment integration
- Support helpdesk with automated email confirmations
- Admin panel for station and session management

---

## Tech Stack

- **Backend:** Python, Flask
- **Database:** MySQL
- **Payments:** Razorpay
- **Email:** Flask-Mail (Gmail SMTP)
- **Maps:** Leaflet.js
- **Frontend:** Jinja2 templates, HTML/CSS
